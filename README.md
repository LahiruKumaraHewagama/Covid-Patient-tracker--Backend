# Covid Patient tracker -Backend
SE Project

## Guide

First clone this project directory.

```bash
git clone https://github.com/SEP-G11/Covid-Patient-tracker--Backend.git
```

Install Node Modules

```bash
npm install
```

 Then run the express server.

```bash
npm start
```

### Api Endpoints
